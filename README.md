## Building the UI

The UI is written in React/Next.js. Before building it, you will need to install [Node](https://nodejs.org/), then [Yarn](https://yarnpkg.com/). After that you can install the dependencies:

```sh
yarn install
```

...than build from source:

```sh
yarn build
```
